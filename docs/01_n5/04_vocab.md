# JLPT N5 어휘 800 — 의미 카테고리별 완전 자료

## 구성 원칙
- 공식 N5 출제 범위 어휘 약 800개를 14개 의미 카테고리로 묶음.
- 각 어휘: 일본어 표기 / 가나 / 한국어 의미 / 품사 / 예문 1개 / 한국어 화자 주의점.
- 의미 카테고리화는 서술 기억(declarative memory)의 의미망 강점을 활용 (소스 01 §2.1).
- 듀얼 코딩(소스 01 §1.5): 한자가 있는 어휘는 한자 + 가나 동시 표기.
- 키워드 기억법(소스 01 §1.6): 어려운 어휘는 한국어 유사음 키워드 부여.
