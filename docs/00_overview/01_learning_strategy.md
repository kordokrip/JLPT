# 학습 전략 문서 — 뇌과학·학습과학 기반 성인 일본어 학습 원칙

## 1. 핵심 학습 원리 7가지 (모든 활동의 근거)

### 1.1 Spaced Repetition (분산 반복)
- **근거**: Kim & Webb (2022, *Language Learning*) 메타분석 — 48개 실험·98개
  효과크기에서 분산 반복이 집중 학습 대비 평균 d ≈ 0.6~0.8의 효과.
  Settles & Meeder (2016, ACL) Half-Life Regression이 망각곡선 직전 재노출의
  효율을 대규모 로그로 입증.
- **적용**: 1일 → 3일 → 7일 → 21일 → 60일 → 120일 누적 리뷰 주기.

### 1.2 Retrieval Practice / Testing Effect (인출 연습 / 시험 효과)
- **근거**: Roediger 계열 100년 누적 효과. 단순 재독 대비 약 2배의 장기 보존.
  Forward Testing Effect — 선행 인출이 후속 학습의 부호화 자체를 강화.
- **적용**: 매 학습일 첫 8분은 "무자료 회상" 워밍업으로 시작.
  자료를 보고 익히는 시간보다 *꺼내는 시간*을 길게.

### 1.3 Interleaving (인터리빙)
- **근거**: Nakata & Suzuki (2019, *Modern Language Journal*),
  Pan et al. (2024) — 유사 항목 변별이 필요한 과제(동사 활용, 비슷한 카나,
  음/훈독)에서 블록 학습보다 인터리빙이 지연 검사에서 유의하게 우세.
- **적용**: 헷갈리는 항목은 반드시 *섞은 순서*로 출제.
  シ/ツ, ソ/ン, ようだ/そうだ/らしい/みたい, ば/たら/と/なら 등.

### 1.4 Desirable Difficulty (바람직한 어려움)
- **근거**: Bjork (1994, 2011) — 정답률 70~85% 구간에서 장기 보존 최대화.
  95% 초과 시 너무 쉬워 보존 효과 미흡, 50% 이하 시 좌절·이탈.
- **적용**: 단원별 정답률 자가 기록 → 95% 초과 단원은 분량 증가,
  60% 미만 단원은 분량 감소·역행 복습.

### 1.5 Dual Coding (이중 부호화)
- **근거**: Paivio 이론. ESL 어휘 연구 (Zhang et al., 2022) — 시각+청각+의미
  동시 부호화 시 회상률 25~40% 상승. 단, 채널 4개 이상은 인지 부하 역효과.
- **적용**: 한 항목당 채널 2~3개 (시각 형태 + 청각 발음 + 의미 이미지).

### 1.6 Mnemonic Keyword Method (키워드 기억법)
- **근거**: Pressley & Levin, Wei & Mayer (2024) 메타분석 — 효과크기 d ≈ 1.0.
  모국어 유사음 단어를 이미지로 연결.
- **적용**: 일본어 단어·한자에 한국어 유사음 키워드 + 시각 이미지 부착.

### 1.7 Sleep-dependent Consolidation (수면 의존 응고화)
- **근거**: Mirković & Gaskell (2016) 메타분석,
  *Psychonomic Bulletin & Review* (2021) — 수면 직전 학습 어휘가 동일 시간
  깨어있는 조건 대비 회상률 유의 우월. SWS(서파수면) 동안 해마→신피질 재활성.
- **적용**: 매일 취침 1~2시간 이내 5분 "오늘의 핵심 5개" 재인출.

## 2. 성인 뇌의 강점·약점

### 2.1 강점
- 분석적 사고와 명시적 규칙 이해 (전전두엽 우세)
- 메타인지 (자기 학습 모니터링)
- 서술 기억(declarative memory) 시스템의 풍부한 의미망
- 모국어와의 대조 분석 능력

### 2.2 약점
- 음운 가소성 저하 (모국어에 없는 음소 변별·발화)
- 절차 기억(procedural memory) 자동화 속도 저하
- 단기 기억 용량은 유지되나 간섭 취약성 증가

### 2.3 전략적 함의
- **명시적 규칙 → 반복 자동화** 순서 (서술→절차 전환)
- **귀를 먼저 만들기**: 문자 학습 전 박·피치 감각 형성
- **모국어 대조 명시화**: "한국어와 무엇이 다른가"를 매 항목 표기

## 3. 하루 60분 학습 블록 표준 템플릿

| 블록 | 시간 | 활동 | 근거 원리 |
|---|---|---|---|
| ① 인출 워밍업 | 8분 | 전날·지난주·지난달 항목 무자료 회상 | 1.2 시험효과 + 1.1 분산반복 |
| ② 분석적 인풋 | 15분 | 신규 항목 원리·구조 이해 | 2.1 서술기억 + 1.5 듀얼코딩 |
| ③ 능동 산출 | 15분 | 쓰기·소리내어 읽기·작문·치환 | 절차기억 전환 + 1.2 시험효과 |
| ④ 변별 인터리빙 | 7분 | 헷갈리는 항목 섞어 풀기 | 1.3 인터리빙 + 1.4 Desirable Difficulty |
| ⑤ 메타인지 | 5분 | 오답 한 줄 요약, 정답률 기록 | 메타인지 |
| ⑥ 취침 전 재인출 | 5분 (별도) | 오늘의 5개 가리고 회상 | 1.7 수면 응고화 |

## 4. 신규/복습 비율 절대 규칙
- 매 학습일 신규 항목 ≤ 30%, 복습 항목 ≥ 70%
- 이 비율은 인지 부하 이론(Sweller)과 SRS 알고리즘의 공통 권장치.

## 5. 정답률 자가 점검 시스템

| 단원 정답률 | 다음 학습일 조치 |
|---|---|
| 95% 초과 | 신규 분량 1.3배 + 심화 도전 5문항 추가 |
| 75~94% | 표준 분량 유지 (이상적 구간) |
| 60~74% | 신규 분량 0.7배 + 해당 단원 복습 1회 추가 |
| 60% 미만 | 신규 학습 중단 + 이전 단원으로 역행 복습 |

## 6. 일본어 한국어 화자 특유 함정 체크리스트

### 6.1 발음
- 장단음 (おじさん/おじいさん, ビル/ビール)
- 촉음 (きて/きって, おと/おっと)
- 요음 (びょういん 병원 / びよういん 미용실)
- 청탁음 (か/が, た/だ, さ/ざ, は/ば/ぱ)
- 피치 액센트 (はし: 다리/젓가락/끝)

### 6.2 문자
- 비슷한 카나 변별: シ/ツ, ソ/ン, ね/れ/わ/ぬ/め, は/ほ, る/ろ, コ/ユ
- 한자 음독/훈독 공존 (生 → せい/しょう/い/う/なま 등)

### 6.3 문법
- 조사 は vs が (한국어 "은/는" vs "이/가"와 부분 일치만)
- 자동사/타동사 페어 (開く/開ける, 始まる/始める)
- 수수 표현 (あげる/もらう/くれる) — 한국어 "주다/받다"보다 복잡
- 조건 4종 변별 (と/ば/たら/なら)
- 추측 4종 변별 (ようだ/そうだ/らしい/みたい)
- 경어 3종 (尊敬語/謙譲語/丁寧語)

## 9. nihongo-n3 앱 FSRS 알고리즘 학습 전략 연동 (2026-05-26 업데이트)

### 9-1. FSRS 현황 및 학습 전략 연관성

nihongo-n3 앱은 현재 **FSRS-5 (W[19])** 알고리즘을 구현하고 있습니다.  
Anki 25.07이 FSRS-6 (W[21])을 기본값으로 채택한 상황에서 업그레이드 예정.

| FSRS 개념 | 학습 전략 원리 | 앱 구현 상태 |
|----------|--------------|------------|
| Stability (안정도) | 1.1 분산 반복 — 간격 최적화 | ✅ 구현됨 |
| Difficulty (난이도) | 1.4 바람직한 어려움 — 70~85% 정답률 구간 유지 | ✅ 구현됨 |
| Again/Hard/Good/Easy 4단계 | 1.2 인출 연습 강도 조절 | ✅ 구현됨 |
| Relearning 상태 | 1.2 재인출 → 재부호화 사이클 | ✅ 구현됨 |
| **학습 단계 설정** | 1.4 단계별 어려움 조절 | ❌ 하드코딩 (1분/10분/1일) |
| **개인 W 최적화** | 개인 망각 패턴 반영 | ❌ 고정 가중치 |

### 9-2. 권장 복습 설정

```
일일 신규 카드 한도: 20~30개 (Settings에서 조절 가능)
이유: 새로운 항목이 많아질수록 '인지 부하' 증가 → 처리 깊이 저하
     초기 2주: 10개 / 적응 후: 20~30개로 점진적 증가 권장

학습 단계 (현재 고정, 향후 설정 가능화 예정):
  1분 → 10분 → 1일 (FSRS 기본 학습 단계)
```

### 9-3. FSRS + 학습 원리 최적 활용법

1. **Again 사용 신중하게** — Again은 `difficulty +0.2` 적용. 지나친 Again 반복 시 카드 난이도가 10에 수렴 → 정답률 저하
2. **Hard 활용** — "알긴 알았는데 시간이 걸렸다" 상황에 사용. stability 보정.
3. **Good 기본값** — 70~85% 신뢰도로 기억한 경우.
4. **Easy 절제** — 확실히 아는 것에만. 너무 많이 쓰면 간격이 너무 벌어져 망각 가능성.
5. **취침 전 복습** — 7절 학습 전략과 연동. 취침 1~2시간 내 "오늘의 핵심 5개" 재인출.
