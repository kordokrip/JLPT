## 0. 개요

| 항목 | 값 |
|---|---|
| 신규 어휘 | 약 3,700개 |
| 누적 어휘 (N5+N4+N3) | 약 800 + 1,500 + 3,700 ≈ 6,000개 |
| 학습 기간 | 16주 |
| 주당 신규 | 약 230개 |
| 일당 신규 | 약 33개 (주 7일 기준) / 약 46개 (주 5일 학습 기준) |
| 학습 시간/단어 | 평균 1.2분 (인풋 25초 + 산출 35초 + 인출 12초 + 인터리빙 단가) |
| 정답률 목표 | 75–85% (Desirable Difficulty 구간) |

### 학습 원리 (소스 01 연계)
- **분산 반복**: 1·3·7·21·60일 주기, Half-Life Regression 적용
- **인터리빙**: 매주 카테고리 2~3개를 섞어 변별 강화
- **듀얼 코딩**: 한자 어휘는 한자(시각) + 음(청각) + 의미(개념) 3채널 동시 부호화
- **키워드 기억법**: 한국어 한자음 유사성 적극 활용 (성호 님 모국어 강점)
- **수면 응고**: 취침 전 5개 어휘 미니 인출

- 한국어 한자음과의 일치율 매우 높음 → 키워드법으로 1주차에 50% 이상 학습 가능

- 인터리빙 트랙: 동음이의 30쌍 따로 5분 트랙

- 중앙: "JLPT N3 어휘 1,850 (전반부)"
