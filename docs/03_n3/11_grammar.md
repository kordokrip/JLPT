## 2. 카테고리 G1 — 추측·양태·전문 (22문형)

### G1-1. ~そうだ (양태) "~할 것 같다"
- **접속**: V[ます형] / い형(어간) / な형(어간) + そうだ
- **의미**: 시각·정황에 근거한 즉각 추측
- **한국어 차이**: ~할 것 같다 (외관 판단)
- **예문**:
  1. このケーキ、美味しそうですね。 (이 케이크 맛있어 보이네요.)
  2. 雨が降りそうだ。 (비가 올 것 같다.)
  3. 彼は元気がなさそうだ。 (그는 기운이 없어 보인다.)
- **오류**: いい → よさそう (× いさそう), ない → なさそう (× なそう)

### G1-2. ~そうだ (전문) "~라고 한다"
- **접속**: 보통형 + そうだ
- **의미**: 들은 정보를 그대로 전달
- **차이**: 양태 そう와 접속 다름! (보통형 vs ます형 어간)
- **예문**:
  1. ニュースによると、明日は雪が降るそうだ。
  2. 田中さんは結婚するそうです。
  3. あのレストランは美味しいそうだ。
- **오류**: 양태 そう와 혼동 (降りそう = 올 것 같다 vs 降るそう = 온다고 한다)

### G1-3. ~ようだ / ~みたいだ "~인 듯하다"
- **접속**: 보통형 + ようだ (な형은 ~な, 명사는 ~の) / 보통형 + みたいだ (회화)
- **의미**: 화자의 주관적 추측·비유
- **한국어 차이**: ~인 듯하다·~같다
- **예문**:
  1. 風邪をひいたようだ。
  2. 彼は学生のようです。
  3. 子供みたいに泣いた。 (비유)
- **오류**: 명사 접속 차이 (学生のようだ ○ / 学生ようだ ×)

### G1-4. ~らしい "~인 듯하다·~답다"
- **접속**: 보통형 + らしい (な·명사는 직접 접속)
- **의미**: ① 객관적 근거 추측 ② 전형성(전형적인 ~답다)
- **차이**: そうだ(전문)보다 정보원 모호, ようだ보다 객관적
- **예문**:
  1. 田中さんは来ないらしい。 (들리는 바로는)
  2. 彼は男らしい人だ。 (남자다운)
  3. 春らしい天気ですね。
- **오류**: みたい·よう와 의미 차이 모호

### G1-5. ~かもしれない "~일지도 모른다"
- **접속**: 보통형 + かもしれない (な·명사는 직접)
- **의미**: 가능성 50% 이하의 추측
- **예문**:
  1. 明日雨が降るかもしれない。
  2. 彼は来ないかもしれません。
- **오류**: ~だろう와 확신도 혼동

### G1-6. ~に違いない "~임에 틀림없다"
- **접속**: 보통형 + に違いない (な·명사는 직접)
- **의미**: 확신도 90% 이상의 추측
- **예문**:
  1. 彼は嘘をついているに違いない。
  2. これは本物に違いない。
- **오류**: ~はずだ와 혼동 (はず는 논리 근거, 違いない는 직감 확신)

### G1-7. ~はずだ "~할 터이다·~할 것이다"
- **접속**: 보통형 + はずだ (な형은 ~な, 명사는 ~の)
- **의미**: 논리·근거 기반 당연한 결론
- **예문**:
  1. 田中さんは今日来るはずだ。 (약속했으니까)
  2. このボタンを押せば動くはずです。
- **오류**: 명사 접속 (学生のはずだ ○)

### G1-8. ~はずがない "~일 리가 없다"
- **접속**: 보통형 + はずがない
- **의미**: 강한 부정 추측
- **예문**:
  1. 彼がそんなことを言うはずがない。
  2. 田中さんが嘘をつくはずがありません。
- **오류**: ~わけがない와 거의 동의 (G1-9 참고)

### G1-9. ~わけがない "~일 리가 없다"
- **접속**: 보통형 + わけがない
- **의미**: 강한 부정 (はずがない와 거의 동의, 화자 감정 강함)
- **예문**:
  1. こんな簡単な問題が解けないわけがない。
- **오류**: わけだ·わけではない와 의미 다름 (G1-10, G1-11)

### G1-10. ~わけだ "~인 셈이다·당연하다"
- **접속**: 보통형 + わけだ (な형 ~な, 명사 ~の·だった)
- **의미**: 앞 정보로부터 도출되는 당연한 결론
- **예문**:
  1. 10年も日本に住んでいるから、日本語が上手なわけだ。
  2. 道理で寒いわけだ。雪が降っている。
- **오류**: はずだ와 혼동 (はず=예측, わけ=납득)

### G1-11. ~わけではない "(반드시) ~인 것은 아니다"
- **접속**: 보통형 + わけではない
- **의미**: 부분 부정 (전면 부정 아님)
- **예문**:
  1. 嫌いなわけではないが、あまり食べない。
  2. 全然できないわけではありません。
- **오류**: わけがない (강한 부정)와 혼동

### G1-12. ~わけにはいかない "~할 수는 없다"
- **접속**: V[사전형] + わけにはいかない
- **의미**: 사회적·도덕적 이유로 불가능
- **예문**:
  1. 仕事を休むわけにはいかない。
  2. 約束したから、行かないわけにはいかない。 (이중부정 = 가야만 한다)
- **오류**: ~ことはできない (능력 불가)와 차이

### G1-13. ~らしい (전형성 강조) "전형적인 ~답다"
- (G1-4와 동일 문형의 다른 용법)
- **예문**: 彼女は本当に女らしい。

### G1-14. ~気がする "~한 기분이 든다"
- **접속**: 보통형 + 気がする
- **의미**: 막연한 직감
- **예문**:
  1. 誰かに見られている気がする。

### G1-15. ~ように見える / ~ように思える
- **의미**: ~처럼 보인다 / 생각된다
- **예문**: 元気なように見える。

### G1-16. ~と思われる "~라고 생각된다"
- **접속**: 보통형 + と思われる
- **의미**: 객관·격식 추측
- **예문**: この事件は人為的なものと思われる。

### G1-17. ~と言われている "~라고 일컬어진다"
- **의미**: 일반 통념
- **예문**: 日本は安全な国だと言われている。

### G1-18. ~とか "~라던가"
- **접속**: 보통형 + とか
- **의미**: 불확실한 전문
- **예문**: 田中さん、結婚するとか。

### G1-19. ~って (회화 전문) "~래"
- **접속**: 보통형 + って
- **의미**: 친한 사이의 인용·전문
- **예문**: 明日休みだって。

### G1-20. ~ようがない "~할 방법이 없다"
- **접속**: V[ます형] + ようがない
- **의미**: 수단·방법의 부재
- **예문**: 連絡先が分からないから、連絡しようがない。

### G1-21. ~に決まっている "~임에 결정돼 있다·당연하다"
- **접속**: 보통형 + に決まっている (な·명사 직접)
- **의미**: 매우 강한 확신
- **예문**: そんなことしたら、怒られるに決まっている。

### G1-22. ~ものだ "~인 법이다"
- **접속**: 보통형 + ものだ
- **의미**: 일반적 진리·당연한 성질
- **예문**: 子供は外で遊ぶものだ。

## 3. 카테고리 G2 — 의무·허가·금지·필요 (15문형)

### G2-1. ~なければならない / ~なければいけない "~해야만 한다"
- **접속**: V[ない형] - い + なければならない
- **의미**: 의무
- **차이**: ならない=일반·도덕적, いけない=개인적
- **예문**:
  1. 明日までにレポートを出さなければならない。
- **오류**: N4 복습, 회화 축약 ~なきゃ / ~なくちゃ

### G2-2. ~なくてはならない / ~なくてはいけない
- 동의어. 회화 ~なくちゃ

### G2-3. ~ないといけない "~하지 않으면 안 된다"
- **의미**: 회화체 의무
- **예문**: 早く起きないといけない。

### G2-4. ~べきだ "~해야 한다"
- **접속**: V[사전형] + べきだ (する → するべき/すべき 둘 다 가능)
- **의미**: 도덕·당위
- **차이**: なければならない(외적 강제)보다 화자 의견 강함
- **예문**:
  1. 約束は守るべきだ。
  2. 学生は勉強すべきだ。
- **오류**: べきではない = ~해서는 안 된다

### G2-5. ~べきではない "~해서는 안 된다"
- **예문**: 嘘をつくべきではない。

### G2-6. ~てもいい "~해도 된다"
- N5 복습. N3에서는 격식 ~てもよろしい로 확장

### G2-7. ~てもかまわない "~해도 상관없다"
- **의미**: ~ていい보다 양보·허용 뉘앙스
- **예문**: タバコを吸ってもかまいません。

### G2-8. ~ても差し支えない "~해도 무방하다"
- **격식**: 공문서·격식 회화

### G2-9. ~てはいけない "~해서는 안 된다"
- N5 복습. 회화 축약 ~ちゃいけない / ~じゃいけない

### G2-10. ~てはならない (격식 금지)
- **예문**: 嘘をついてはならない。

### G2-11. ~ことはない "~할 필요는 없다"
- **접속**: V[사전형] + ことはない
- **의미**: 불필요·위로
- **예문**: そんなに心配することはない。
- **오류**: 명사 + ではない(부정)와 혼동

### G2-12. ~必要がある / ~必要はない
- **예문**: 予約する必要があります。

### G2-13. ~ざるを得ない "~하지 않을 수 없다"
- **접속**: V[ない형] - い + ざるを得ない (する → せざるを得ない)
- **의미**: 어쩔 수 없는 의무
- **예문**: 規則だから、従わざるを得ない。
- **오류**: する→せざる (×しざる) — 빈출 함정

### G2-14. ~しかない / ~するしかない "~할 수밖에 없다"
- **접속**: V[사전형] + しかない
- **의미**: 다른 선택지 없음
- **예문**: 電車が止まったから、歩くしかない。

### G2-15. ~よりほかない "~할 수밖에 없다"
- 동의: しかない (격식체)

## 4. 카테고리 G3 — 의지·희망·계획·예정 (14문형)

### G3-1. ~(よ)うと思う "~하려고 한다"
- **접속**: V[의지형] + と思う
- **의미**: 화자의 의지·계획
- **예문**: 来年、留学しようと思っている。
- **오류**: ~ようと思っている(지속적 의지) vs ~ようと思う(현재 결정)

### G3-2. ~(よ)うとする "~하려고 하다"
- **의미**: 시도·직전 동작
- **예문**: 寝ようとしたとき、電話が鳴った。

### G3-3. ~つもりだ "~할 작정이다"
- N4 복습. N3 확장: ~つもりはない (~할 생각 없다), ~つもりだった (~할 작정이었다)

### G3-4. ~つもりで "~할 생각으로"
- **예문**: 死ぬつもりで頑張った。 (각오)

### G3-5. ~予定だ "~할 예정이다"
- **접속**: V[사전형] / 명사+の + 予定だ
- **예문**: 来月、出張する予定です。

### G3-6. ~ことにする "~하기로 하다 (결정)"
- **접속**: V[사전형/ない형] + ことにする
- **의미**: 화자의 결정
- **예문**: タバコをやめることにした。

### G3-7. ~ことになる "~하게 되다 (외부 결정)"
- **의미**: 자기 의지 외 결정
- **예문**: 来年、東京に転勤することになりました。
- **차이**: ことにする(자기) vs ことになる(외부)

### G3-8. ~ことにしている "~하기로 하고 있다 (습관)"
- **의미**: 자기 결정으로 지속하는 습관
- **예문**: 毎朝、走ることにしている。

### G3-9. ~ことになっている "~하기로 되어 있다 (규칙)"
- **의미**: 규칙·관습
- **예문**: 9時までに来ることになっている。

### G3-10. ~たい / ~たがる "~하고 싶다 / ~하고 싶어 하다"
- N5 복습. N3 확장: 3인칭은 ~たがる

### G3-11. ~てほしい "~해 주었으면 한다"
- **접속**: V[て형] + ほしい
- **의미**: 타인에 대한 희망
- **예문**: 早く来てほしい。
- **오류**: ~たい(자기 행위) vs ~てほしい(상대 행위)

### G3-12. ~ように "~하도록"
- **접속**: V[사전형/ない형] + ように
- **의미**: 목표·간접 의지
- **예문**: 忘れないように、メモした。

### G3-13. ~ようにする "~하도록 하다"
- **의미**: 노력하다
- **예문**: 毎日運動するようにしています。

### G3-14. ~ようになる "~하게 되다 (능력·습관 변화)"
- **예문**: 日本語が話せるようになった。

## 5. 카테고리 G4 — 원인·이유·근거 (18문형)

### G4-1. ~から / ~ので "~니까 / ~기 때문에"
- N5 복습. ので가 객관·격식

### G4-2. ~ために "~때문에"
- **접속**: 보통형 + ため(に) (な ~な, 명사 ~の)
- **의미**: 원인 (격식)
- **예문**: 雨のために、試合が中止になった。
- **오류**: 목적 ために(G7)와 동음, 접속 다름

### G4-3. ~おかげで "~덕분에"
- **접속**: 보통형 + おかげで (な ~な, 명사 ~の)
- **의미**: 긍정적 원인
- **예문**: 先生のおかげで合格できた。

### G4-4. ~せいで "~탓에"
- **의미**: 부정적 원인
- **예문**: 寝坊したせいで、遅刻した。
- **오류**: おかげで와 감정 부호 정반대

### G4-5. ~せいか "~탓인지"
- **의미**: 불확실한 원인 추측
- **예문**: 疲れているせいか、頭が痛い。

### G4-6. ~おかげか / ~おかげさまで
- 회화 빈출

### G4-7. ~によって "~에 의해서"
- **접속**: 명사 + によって
- **의미**: ① 원인·이유 ② 수단·방법 ③ 기준
- **예문**:
  1. 地震によって、多くの家が壊れた。 (원인)
  2. 国によって習慣が違う。 (기준)
- **다의어 주의**

### G4-8. ~ことから "~로부터·~점에서"
- **의미**: 근거
- **예문**: この花がよく咲くことから、「桜の名所」と呼ばれている。

## 14. 카테고리 G13 — 인터리빙 변별 챕터 (10세트)

> **목적**: 의미가 유사한 문형을 직접 대조하여 변별력 강화. **JLPT N3 합격선 결정 요소**.

### G13-1. ように vs ために (목적)

| 항목 | ように | ために |
|---|---|---|
| 주체 | 무의지·자동사·가능 | 의지·동일 주체 |
| 의미 | 자연 발생 목적 | 의도된 목적 |
| 예문 | 聞こえるように大声で話す | 合格するために勉強する |

### G13-2. と vs ば vs たら vs なら (조건)

| 항목 | と | ば | たら | なら |
|---|---|---|---|---|
| 의미 | 자연 결과 | 가정·일반 | 1회·완료 후 | 화제 한정 |
| 명령형 후속 | ✗ | ✗ | ○ | ○ |
| 예문 | 春になると桜が咲く | お金があれば買う | 帰ったら電話して | 行くなら早く |

### G13-3. はず vs わけ vs べき (당위·추측)

| 항목 | はず | わけ | べき |
|---|---|---|---|
| 의미 | 논리 예측 | 납득·결론 | 도덕 당위 |
| 예문 | 来るはずだ | 上手なわけだ | 守るべきだ |

### G13-4. らしい vs ようだ vs みたい vs そう

| 형 | 정보원 | 객관도 |
|---|---|---|
| そうだ(전문) | 명확 출처 | 객관 |
| らしい | 들은 정보 | 객관 |
| ようだ | 자기 관찰 | 주관 |
| みたい | 자기 관찰(구어) | 주관 |
| そうだ(양태) | 시각 |  주관 |

### G13-5. ところだ vs ばかりだ (직후)

| 항목 | ところだ | ばかりだ |
|---|---|---|
| 시점 | 객관·실제 | 주관·심리 |
| 예문 | 今、着いたところ | さっき食べたばかり |

### G13-6. のに vs くせに vs にもかかわらず (역접)

| 항목 | のに | くせに | にもかかわらず |
|---|---|---|---|
| 뉘앙스 | 의외·불만 | 비난·경멸 | 격식 |
| 주체 | 자유 | 주어 동일 | 자유 |

### G13-7. おかげで vs せいで vs ために (원인)

| 항목 | おかげで | せいで | ために |
|---|---|---|---|
| 감정 | 긍정 | 부정 | 중립·격식 |

### G13-8. うちに vs 間に (시간 구간)

| 항목 | うちに | 間に |
|---|---|---|
| 의미 | 상태 변하기 전 | 시간 구간 내 |
| 예문 | 若いうちに | 寝ている間に |

### G13-9. ばかり vs だけ vs しか (한정)

| 항목 | ばかり | だけ | しか~ない |
|---|---|---|---|
| 뉘앙스 | 빈도·과다 | 단순 한정 | 한정+부족·아쉬움 |
| 예문 | テレビばかり見る | 私だけ知っている | 千円しかない |

### G13-10. ように vs ようになる vs ようにする

| 항목 | ように | ようになる | ようにする |
|---|---|---|---|
| 의미 | 목적·간접의지 | 변화 결과 | 노력·습관 |
| 예문 | 忘れないように | 話せるようになった | 運動するようにしている |

## 20. 학습 팁

### 20-1. 문형을 함수 시그니처로 도식화
```
~ようになる :: VerbDictionary → Change(Ability/Habit)
~ことにする :: VerbDictionary → Decision(Self)
~ことになる :: VerbDictionary → Decision(External)
~ように      :: VerbDictionary → Goal(Indirect)
~ために      :: VerbDictionary → Goal(Direct)
```

### 20-2. 조건문 4종을 상태머신으로
- と: deterministic transition (자연법칙)
- ば: conditional branch (가정)
- たら: sequential post-condition (1회성)
- なら: topic-scoped condition (한정)

### 20-3. 경어 시스템을 권한 모델로
- 존경 = elevated (대상 ↑)
- 겸양 = lowered (자기 ↓)
- 정중 = formalized (관계 격식화)
- → ABAC(Attribute-Based Access Control)과 유사한 사고

### 20-4. 인터리빙을 단위 테스트로
- G13의 10세트를 unit test suite로 간주
- 각 문형 = function under test
- 변별 카드 = assertion (입력 동일, 기대 출력 다름)
- 정답률 90% 이상 = test pass
