# JLPT N4 추가 한자 181자 완전 자료

## 구성 원칙
- N5에서 학습한 103자 위에 **추가**되는 181자. 누적 총 284자.
- N5와 결합하여 합성어를 형성하는 한자가 다수 → "결합 학습" 우선.
- 각 한자: 부수 / 음독 / 훈독 / 의미 / 한국 한자음 / N4 어휘 묶음 / N5 한자와의 합성 예.
- 의미 카테고리 10개로 묶음 (소스 01 §2.1 서술기억 의미망).
- 학습 12주에 걸쳐 분배 (소스 01 §1.1 분산 반복).
- 음/훈독 변별 함정 한자는 별도 인터리빙 챕터.
