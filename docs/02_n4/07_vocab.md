# JLPT N4 어휘 1500 — 의미 카테고리별 완전 자료

## 구성 원칙
- N5 800에서 N4 1500으로 약 700개 신규. 총 누적 1500.
- N5의 14개 카테고리를 그대로 유지하되, 각 카테고리에 N4 어휘를 추가.
- 본 문서는 **N4 신규 700개를 카테고리별로 정리**. N5 어휘는 소스 04 참조.
- 동사는 **자/타 페어** 명시 (N4 핵심 함정).
- 한국어 화자 함정·주의점을 매 어휘에 표기.
