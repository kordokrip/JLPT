# 발음·박(モーラ)·카나 마스터 자료

## PART A. 박(モーラ) 감각 — 일본어 리듬의 기본

### A.1 박이란 무엇인가
- 일본어는 *모라 박자 언어(mora-timed)*. 한국어의 음절과 다름.
- 1모라 = 1박. 동일한 시간 길이로 발화.
- 예: 「東京」= と・う・きょ・う = 4박 (한국어 화자는 흔히 "도쿄" 2박으로 발음 → 화석화 원인)

### A.2 모라 단위 예시

| 일본어 | 모라 분해 | 박수 | 한국어 화자 실수 |
|---|---|---|---|
| 学生 がくせい | が・く・せ・い | 4 | 3박으로 단축 |
| 先生 せんせい | せ・ん・せ・い | 4 | 3박으로 단축 |
| 切手 きって | き・っ・て | 3 | 촉음 무시 |
| 病院 びょういん | びょ・う・い・ん | 4 | 요음·장음 동시 단축 |
| お母さん おかあさん | お・か・あ・さ・ん | 5 | 장음 무시 |

### D.2 혼동쌍 변별 페이지 (인터리빙 전용)

| 혼동쌍 | 변별 포인트 |
|---|---|
| シ vs ツ (카타카나) | シ는 아래에서 위로, ツ는 위에서 아래로 |
| ソ vs ン (카타카나) | ソ는 위에서 아래로(짧), ン은 아래에서 위로(곡선) |
| ね vs れ vs わ vs ぬ vs め | 우측 곡선의 형태와 끝맺음 차이 |
| は vs ほ | ほ는 가로획 하나 더 |
| る vs ろ | る는 끝에 작은 고리, ろ는 없음 |
| き vs さ | き는 가로획 2개, さ는 1개 |
| コ vs ユ (카타카나) | コ는 ㄷ 모양, ユ는 가로획 2개 |
